# Autisme Diffusion - Catalogue Intelligent avec IA

🧩 Application Streamlit pour gérer et explorer le catalogue de produits Autisme Diffusion avec des fonctionnalités d'intelligence artificielle.

## 📋 Fonctionnalités

- **Recherche Intelligente** : Recherche sémantique de produits
- **Recommandations** : Suggestions de produits similaires
- **Analyse du Catalogue** : Visualisations et statistiques
- **Assistant Conversationnel** : Posez des questions en langage naturel
